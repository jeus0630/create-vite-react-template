# Vite React Template! 
A working-level react template based on vite bundling tool.

## Installation and Usage
1. Install libraries using yarn
    ```zsh
    yarn
    ```
2. Install Husky 
    ```zsh
    yarn prepare
    ```
3. Enjoy your coding!
